# Sort
Testprogram för sortering utav siffror.

Siffrorna ligger i en vector som skickas till de olika sorteringalgortimerna och tidtas för att se vilken som är snabbast.

Sorteringsalgortimer:
-Dream /Albin
-Quick /Fritjof
